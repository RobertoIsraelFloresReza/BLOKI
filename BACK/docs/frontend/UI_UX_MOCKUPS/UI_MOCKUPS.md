## Design System

### Colors

```
Primary Blue:    #7B16FF (Stellar Blue)
Secondary Purple: #9945FF (Stellar Purple)
Accent Cyan:     #00D1FF (Stellar Cyan)
Success Green:   #10B981
Warning Yellow:  #F59E0B
Error Red:       #EF4444
Gray Scale:      #111827, #374151, #6B7280, #9CA3AF, #D1D5DB, #F3F4F6
```

### Typography

```
Headings:   Inter, -apple-system, BlinkMacSystemFont, sans-serif
Body:       Inter, -apple-system, BlinkMacSystemFont, sans-serif
Monospace:  'Courier New', Courier, monospace (for addresses/hashes)
```

### Spacing

```
xs:  0.25rem (4px)
sm:  0.5rem (8px)
md:  1rem (16px)
lg:  1.5rem (24px)
xl:  2rem (32px)
2xl: 3rem (48px)
```

## Responsive Breakpoints

```
Mobile:   < 640px
Tablet:   640px - 1024px
Desktop:  > 1024px
```

All mockups should stack vertically on mobile, use 2-column grid on tablet, and 3-column grid on desktop.

## Accessibility Notes

- All interactive elements have focus states
- Minimum touch target size: 44x44px
- Color contrast ratio: minimum 4.5:1 for text
- All icons have aria-labels
- Keyboard navigation supported throughout
- Screen reader friendly with proper ARIA attributes
